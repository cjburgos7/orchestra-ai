# Orchestra World Design Skill

Orchestra is NOT a generic SaaS platform.

It is:
- a cinematic startup-world generation engine
- an emotionally intelligent orchestration system
- a premium founder experience
- a living ecosystem of startup worlds

## Visual Philosophy

Every page should feel:
- cinematic
- immersive
- emotionally intelligent
- atmospheric
- spatial
- luxurious
- orchestral
- editorial
- futuristic
- calm but ambitious

## Core Aesthetic

Blend:
- Interstellar
- Apple cinematic product films
- A24 atmosphere
- luxury editorial websites
- unseen studio
- futuristic architecture
- bioluminescent environments
- heavenly dreamlike worlds
- orchestral motion systems

## Layout Rules

Avoid:
- generic SaaS layouts
- repetitive cards
- dashboard energy
- flat sections
- template composition
- startup clichés
- crypto aesthetics

Prefer:
- asymmetrical composition
- giant image-driven sections
- immersive viewport storytelling
- cinematic environmental heroes
- floating UI systems
- restrained interfaces
- editorial typography
- atmospheric layering
- emotional pacing

## Typography

Typography should feel:
- giant
- elegant
- editorial
- cinematic
- emotionally spacious

Mix:
- refined serif display fonts
- modern sans-serif systems

Avoid:
- overly techy fonts
- startup bro aesthetics

## Motion Philosophy

Use Framer Motion extensively.

Motion should feel:
- fluid
- orchestral
- cinematic
- weightless
- atmospheric
- immersive

Prefer:
- floating motion
- staggered reveals
- slow environmental parallax
- cinematic fades
- layered transitions
- soft hover animations
- ambient movement

Avoid:
- aggressive motion
- gimmicky animation
- excessive bounce effects

## Image Philosophy

Images are CRITICAL.

Every generated world must use:
- category-relevant imagery
- cinematic environmental storytelling
- emotionally immersive visuals
- premium composition

Never:
- mismatch images to categories
- reuse generic assets
- use irrelevant stock photos

Worlds should feel:
- alive
- believable
- cinematic
- emotionally distinct

## Orchestra World Principle

Every startup should feel like:
its own living universe with a unique emotional atmosphere, visual culture, and cinematic identity.
